Comands:

cd <mapes url> - dodas uz šo mapi;
.. - iziet ārā no pašreizējās mapes;
mkdir <name> - izveido jaunu mapi;
touch <name> - izveidot failu mapē;
code . - atvēr visual studio code;


GitHub comands:

git init - izveido jaunu .git repository datorā;
git remote add origin <repozitory link> - savienojas ar tavu GitHub repozitoriju;
git branch -M main - pārveido branch no Master uz main;
git add . - pievieno veiktās izmaiņas;
git commit -m "<commit name>" - pievieno izmaiņām nosaukumu;
git push - ievieto izmaiņas GitHub repozitorijā (nav pilnā nosaukuma, tomēr var atrast ierakstot šo komandu un tad nokopējot no padomiem);
